# ai-sql-analytics-engine
AI SQL Analytics Engine
Natural-Language → SQL → Insights Automation with Python, SQLite & Tableau
This project implements an AI-driven analytics engine where a user can ask questions in normal English, and the system automatically:
Generates the corresponding SQL query
Executes it on a real SQLite database
Exports the results as CSV
Feeds results into Tableau dashboards
Enables automated business insights with zero manual SQL writing
This project simulates the workflow of an AI-powered Data Analyst, combining Python, AI-inspired logic, SQL, and Tableau dashboards.

**Tech Stack**
Backend
Python
SQLite (built-in database)
Pandas
SQLAlchemy (optional)
AI Layer
Rule-based NL → SQL generator
Upgrade-ready for OpenAI / Gemini / Llama
Visualization
Tableau Desktop
Tableau Dashboard with filters, KPIs & advanced layout
